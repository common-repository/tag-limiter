A Wordpress Plugin which allows you to control the maximum number of tags shown in the Tag Cloud,  through the Wordpress Dashboard.

== Description ==
WP LimitTags is a simple Wordpress plugin which allows you to control the maximum number of Tag Cloud in your Wordpress Tag Cloud widget. So you will get rid of the hassle of huge Tag Cloud sizes when your website grows.

You can simply install this plugin and Activate it and then control the number of tags shown in the Tag Cloud.

== Installation ==
1. Install the Plugin
2. Activate the Plugin.
3. Goto the Admin Dashboard and find WP Limit tags, under Settings
